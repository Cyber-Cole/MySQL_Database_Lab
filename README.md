<h1>MySQL Database Lab</h1>


<h2>Description</h2>
I used MySQL to create a small database and then perform SQL queries to verify the database was functional. The database consists of different fictional coffee shops, the employees of those coffee shops, the suppliers for the coffee shops, and the types of coffee sold by each shop.
<br />


<h2>Languages and Utilities Used</h2>

- <b>SQL</b> 
- <b>MySQL Workbench 8.0</b>

<h2>Environments Used </h2>

- <b>Windows 10</b>
